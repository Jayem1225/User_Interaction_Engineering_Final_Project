# UX Optimized Calculator

This was my final project I developed for my User Interaction Engineering course back in 2017.

The intention of the project was to do a user study on an existing piece of software, and develop some improvements based on user feedback and study observations. I chose to find improvements for the Windows Calculator application. 

In order to gather data, several users were live recorded (both the subject and their screen) and given a variety of tasks to complete which ranged from rudimentary addition/subtraction to complex (trig, unit conversion, etc.). Their tasks were provided by and feedback was recorded to Google Forms. The recordings were analyzed to guage how frustrated or satisfied users were with the process of task completion, how long each user took to do each task, and how well the users were able to navigate and use the tool to complete the task.

The final product was developed using WinForms on Visual Studio Code. As of this commit, the only modifications of made to this project from the original, is the introduction of this README.md and .gitignore files.
